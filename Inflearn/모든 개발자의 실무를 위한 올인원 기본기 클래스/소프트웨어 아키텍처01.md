# 소프트웨어 아키텍처

```
출처 : 모든 개발자의 실무를 위한 올인원 기본기 클래스
```

​	

## 소프트웨어 아키텍처란?

소프트웨어의 전체적인 구조를 잡아주는 설계도

- 시스템에 규칙을 만듦
  - 일관적인 코드
  - 모듈 분리와 추상화에 대한 고민이 줄어듬
  - 코드 파악이 빨라짐
- 테스트가 쉬워짐



## 트레이드오프

- 초반에 아키텍처 설계에 대한 시간과 노력 필요
- 알아야 할 규칙이 늘어남
- 팀원 전체가 익숙해져야 함
- 일반적으로 작성해야 하는 코드의 양이 늘어남
