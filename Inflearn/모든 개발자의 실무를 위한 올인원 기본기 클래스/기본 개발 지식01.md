# 기본 개발 지식 : 프로세스 & 스레드

```
출처 : 모든 개발자의 실무를 위한 올인원 기본기 클래스
```



## 프로세스 (Process)

```
실행되고 있는 프로그램
```

메모리에 올라가 OS에 의해 CPU, 메모리 자원을 할당받고 있는 프로그램



[메모리 영역]

- `Code` : 프로그램의 코드
- `Data` : **정적 데이터** (전역변수, 정적변수, 배열, 구조체 등)

- `Stack` : **함수** 데이터 (실행 정보, 지역변수, 파라미터 등) <= 컴파일 때 크기가 결정됨
  - 프로세스를 실행하면서 할당된 Stack의 크기보다 초과되는 데이터가 저장되는 경우 Stack Overflow 발생
- `Heap` : **동적 데이터** (런타임 중 처리)



## 스레드 (Thread)

```
프로세스 안에서 실제로 코드를 실행하는 단위
```

CPU 사용의 기본 단위 => 한 개의 스레드는 한 개의 CPU 코어에 할당됨

프로세스가 할당받은 Stack 메모리를 사용해서 실행

=> `Stack`은 **각 스레드 마다 개인적으로 소유** & `Code`, `Data`, `Heap`은 **프로세스 내에서 공유**



일반적으로 프로그램은 1개의 프로세서 & 1개의 메인 스레드를 사용

But, 필요에 따라 멀티 프로세스, 멀티 스레드 사용이 가능
